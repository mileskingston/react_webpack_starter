# React and webpack starterpoint

A starter point for creating a React app using Webpack 4 and Babel.

## How to run

first install node_modules folder.
```sh
npm install
```

to run development build with local server
```sh
npm run start
```

For production build
```sh
npm run build
```

### Versions

| Plugin | Version |
| ------ | ------ |
| React | 16.3.1 |
| Webpack | 4.5.0 |
